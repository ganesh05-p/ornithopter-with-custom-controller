# 🐦🕹️ Ornithopter with Custom Made Controller

A bio-inspired flying robot designed to replicate flapping-wing flight dynamics using a custom-built wireless transmitter-receiver system. This project demonstrates wireless flight control, embedded systems, and mechanical wing-flapping mechanisms, serving as a foundation for future research in autonomous aerial robotics and surveillance applications.

Inspired by the natural flight of birds, the system features a custom-designed wireless controller and a dual-motor mechanism. One motor drives the wing-flapping mechanism using a gear system, while another motor controls directional movement. The project demonstrates the core mechanical and wireless communication principles required for future ornithopter development.
